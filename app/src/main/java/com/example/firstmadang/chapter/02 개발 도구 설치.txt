            02-1 안드로이드 스튜디오 설치

안드로이드 스튜디오 공식 사이트에서 설치를 한다.


    조언

    폴더 경로에 한글이 포함되면 제대로 실행되지 않을 수도 있다.

    안드로이드 스튜디오 실행 시 메모리 용량을 늘릴 필요가 있다.


    조언

    안드로이드 플랫폼 - 버전 별로 만들어진 실행 환경


설정 화면에서 오른쪽 하단에 Configure 에서 [SDK Platforms] 를 선택한다.
두 번째 탭 [SDK Tools] 를 누르면 설치된 라이브러리나 도구들이 보이는데
'Android SDK Build-Tools, Android Emulator, Android SDK Platform-Tools, Android SDK Tools'
항목들이 체크되었는지 확인한다.


    조언

    안드로이드 최신 플랫폼이 출시되었다면 설정화면 [Configure] 에서 [Check for Update] 로 확인할 수 있다.

    최신 플랫폼은 [Configure] 메뉴에서 [SDK Manager] 에서 최신 버전 플랫폼을 확인하고 설치한다.
