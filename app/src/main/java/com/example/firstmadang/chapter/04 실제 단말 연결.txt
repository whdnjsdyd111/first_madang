            04-1 PC 에 드라이버 설치

단말기는 PC 외부 장치와 연결되는 것으로 드라이브러를 설치해야 한다.

삼성 단말기 https://www.samsungsvc.co.kr/online/downLoadMain.do?isDownload=S

설치받은 드라이버를 설치한다.


            04-2 단말 연결 후 설정

드라이버 설치 후 USB 케이블로 단말과 PC 를 연결하고 안드로이드 스튜디오 오른쪽의 [Device File Explorer]
메뉴를 선택하여 연결된 단말 중 하나를 선택한다. 이를 보이기 위해선 개발자 모드에서 설정을 해야한다.


        삼성 단말에서 개발자 모드 설정

단말에서 설정 창에 '디바이스 정보' 에 들어간다.
'빌드 번호' 를 여러번 터치하여 비밀번호를 입력 후 '개발자 모드' 에 들어간다.

BACK 버튼으로 이전 화면에서 '개발자 옵션' 이 나타났다면 해당 메뉴로 들어간다.
여기서 'USB' 디버깅 항목을 찾아 터치한다.

확인 버튼을 누르면 단말 설정이 끝난다.

이제 안드로이드 디바이스 모니터에 Devices 영역에 새 단말이 표시된 것을 볼 수 있다.
해당 단말기를 선택하여 앱을 실행해보자.
